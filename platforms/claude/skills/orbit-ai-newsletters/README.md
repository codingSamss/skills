# orbit-ai-newsletters

## 作用
AI 新闻简报技能，多源抓取去重排序生成每日摘要，保存到 Obsidian Vault。

## 平台支持
- Claude Code

## 依赖
- orbit-os（共享 Vault 配置）
- Bird CLI（可选，X/Twitter 扩展源）
- Composio MCP（可选，Reddit 扩展源）

## 配置
```bash
./setup.sh orbit-ai-newsletters
```

## 验证
确认 Vault 下存在 `05_资源/Newsletters/` 目录。

## 使用方式
```
/orbit-ai-newsletters
```
