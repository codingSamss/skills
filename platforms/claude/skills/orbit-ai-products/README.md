# orbit-ai-products

## 作用
AI 产品发布追踪技能，多源抓取过滤排序生成每日摘要，保存到 Obsidian Vault。

## 平台支持
- Claude Code

## 依赖
- orbit-os（共享 Vault 配置）
- Bird CLI（可选，X/Twitter 扩展源）
- Composio MCP（可选，Reddit 扩展源）

## 配置
```bash
./setup.sh orbit-ai-products
```

## 验证
确认 Vault 下存在 `05_资源/产品发布/` 目录。

## 使用方式
```
/orbit-ai-products
```
